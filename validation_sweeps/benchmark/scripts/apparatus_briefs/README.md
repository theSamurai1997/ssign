# Apparatus briefs (Phase 0b machinery answer key)

One brief per secretion-system type. Each lists the **canonical machinery gene
families** that build that system, with the unified/conserved gene name and the
common system-specific synonyms a paper might use.

Purpose: brief the per-instance curation agent (step 3.4) so it knows what counts
as *apparatus* (structural/export machinery) versus *cargo* (the secreted effector,
which is NOT machinery and must not be listed). The agent records only gene names
that a cited paper assigns to the apparatus of the specific system instance, with a
verbatim quote and DOI.

These briefs describe gene **families**, not a fixed gene list: a given instance may
lack some accessory components or carry extra ones. The agent reports what the
literature for *that instance's genome* actually documents, using these names as the
recognition vocabulary.

Scope note: T5SS is excluded from the benchmark (autotransporter passenger is part of
the protein itself, no separate apparatus to enumerate).
