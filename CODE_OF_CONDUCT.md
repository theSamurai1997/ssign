# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc].
By participating in this project (as a contributor, issue reporter, or
reviewer), you agree to abide by its terms.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Scope

Applies to all project spaces: the GitHub repository (issues, pull
requests, discussions, code review), project communications, and any
in-person or virtual events where someone is representing the project.

## Reporting

If you experience or witness behaviour that violates this Code of Conduct,
report it privately to one of:

- **M. Teo Reid** (maintainer): `t.reid25@imperial.ac.uk`
- **Dr. Sonja Billerbeck** (PI): `s.billerbeck@imperial.ac.uk`

All reports are handled confidentially. Enforcement follows the Community
Impact Guidelines from the Contributor Covenant.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
